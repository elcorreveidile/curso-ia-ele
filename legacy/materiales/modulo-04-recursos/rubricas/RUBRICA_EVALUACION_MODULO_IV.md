## 📊 RÚBRICA (5 criterios × 4 niveles = 20 puntos cada uno)

### Criterio 1: Variedad de recursos (20 puntos)

| 4 - Excelente (20) | 3+ tipos de recursos (imagen, audio, mapa mental) integrados creativamente en secuencia didáctica |
| 3 - Satisfactorio (15-17) | 3 tipos de recursos presentes con integración adecuada |
| 2 - Básico (10-13) | 2 tipos de recursos con integración básica |
| 1 - Insuficiente (0-8) | Solo 1 tipo de recurso o sin integración |

### Criterio 2: Calidad individual de recursos (20 puntos)

| 4 - Excelente (20) | Todos los recursos son excelentes: auténticos, culturalmente ricos, nivel perfecto, listos para usar |
| 3 - Satisfactorio (15-17) | Recursos mayormente de buena calidad con áreas menores de mejora |
| 2 - Básico (10-13) | Recursos aceptables pero requieren adaptaciones significativas |
| 1 - Insuficiente (0-8) | Recursos pobres o con problemas importantes |

### Criterio 3: Coherencia didáctica (20 puntos)

| 4 - Excelente (20) | Secuencia didáctica completa y coherente: recursos articulados en progresión lógica para unidad |
| 3 - Satisfactorio (15-17) | Recursos conectados en secuencia didáctica mayormente coherente |
| 2 - Básico (10-13) | Recursos sueltos sin conexión clara o secuencia débil |
| 1 - Insuficiente (0-8) | Sin unidad temática o coherencia didáctica |

### Criterio 4: Aplicabilidad real (20 puntos)

| 4 - Excelente (20) | Totalmente implementable con alumnado real el lunes; incluye instrucciones de uso |
| 3 - Satisfactorio (15-17) | Implementable con adaptaciones menores |
| 2 - Básico (10-13) | Requiere varias adaptaciones para ser usable |
| 1 - Insuficiente (0-8) | Difícil o imposible de implementar |

### Criterio 5: Reflexión crítica (20 puntos)

| 4 - Excelente (20) | Análisis profundo: decisiones, iteraciones, aprendizajes, ventajas/limitaciones de IA |
| 3 - Satisfactorio (15-17) | Reflexión adecuada con ejemplos de proceso |
| 2 - Básico (10-13) | Reflexión superficial sin análisis crítico |
| 1 - Insuficiente (0-8) | Sin reflexión o descripción mínima |

## 📝 ESTRUCTURA DE LA ENTREGA

### Parte 1: Kit multimodal (3-5 páginas)

**Imagen:**
- Prompt usado
- Imagen generada
- Cómo se usa en clase (actividad)

**Audio:**
- Prompt usado
- Transcripción del audio
- Duración y nivel
- Cómo se usa en clase

**Mapa mental:**
- Prompt usado (ChatGPT)
- Captura del mapa mental (Coggle/Miro)
- Cómo se usa en clase

### Parte 2: Reflexión (1-2 páginas)

1. **Proceso de creación:** ¿Cuántas iteraciones? ¿Qué cambió entre versiones?
2. **Decisiones:** ¿Por qué elegiste esos prompts específicos?
3. **Ventajas/limitaciones de IA:** ¿Qué hace bien la IA? ¿Qué no hace bien?
4. **Aplicabilidad:** ¿Cómo usarás este kit con tu alumnado real?
