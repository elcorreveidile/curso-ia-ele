# CRITERIOS DE CALIDAD PARA RECURSOS GENERADOS CON IA
### Imágenes, Audio y Mapas Mentales

---

## 1. IMÁGENES

### Criterio 1: Representación cultural auténtica
**✅ Excelente:** Representa la diversidad del mundo hispanohablante (España, Latinoamérica, diferentes regiones)
**✅ Bueno:** Representa al menos 2 países/regiones diferentes
**⚠️ Aceptable:** Representa una región pero de forma auténtica
**❌ Problemático:** Estereotipos ofensivos o representación inexacta

### Criterio 2: Adecuación al nivel MCER
**✅ Excelente:** Perfectamente alineado con nivel (simple para A1, rico en detalles para B2+)
**✅ Bueno:** Mayoritariamente alineado con nivel
**⚠️ Aceptable:** Requiere pequeñas adaptaciones
**❌ Problemático:** No corresponde al nivel declarado

### Criterio 3: Calidad técnica
**✅ Excelente:** Alta resolución, buena composición, colores apropiados
**✅ Bueno:** Calidad técnica adecuada
**⚠️ Aceptable:** Calidad suficiente pero no óptima
**❌ Problemático:** Borrosa, mal compuesta, colores distorsionados

### Criterio 4: Relevancia didáctica
**✅ Excelente:** Genera múltiples actividades posibles (descripción, narración, debate)
**✅ Bueno:** Útil para 1-2 actividades específicas
**⚠️ Aceptable:** Usable con modificaciones
**❌ Problemático:** Difícilmente usable en clase

### Criterio 5: Ausencia de sesgos
**✅ Excelente:** Sin sesgos evidentes de género, raza, clase social
**✅ Bueno:** Sesgos mínimos
**⚠️ Aceptable:** Algunos sesgos pero no ofensivos
**❌ Problemático:** Sesgos ofensivos o estereotipos dañinos

---

## 2. AUDIOS

### Criterio 1: Acento y pronunciación
**✅ Excelente:** Acento español nativo (peninsular o latinoamericano según contexto), pronunciación clara
**✅ Bueno:** Acento mayoritariamente correcto con pequeños errores
**⚠️ Aceptable:** Acento aceptable pero no nativo
**❌ Problemático:** Acento incorrecto o pronunciación poco clara

### Criterio 2: Velocidad y ritmo
**✅ Excelente:** Velocidad adecuada al nivel MCER, pausas naturales, ritmo fluido
**✅ Bueno:** Velocidad mayormente adecuada
**⚠️ Aceptable:** Velocidad requiere ajustes
**❌ Problemático:** Demasiado rápido o lento para el nivel

### Criterio 3: Claridad del sonido
**✅ Excelente:** Sonido limpio sin ruido de fondo, volumen consistente
**✅ Bueno:** Buena calidad con mínimas imperfecciones
**⚠️ Aceptable:** Calidad aceptable
**❌ Problemático:** Ruido, eco, volumen inconsistente

### Criterio 4: Contenido lingüístico
**✅ Excelente:** Vocabulario y estructuras perfectamente alineados con nivel MCER
**✅ Bueno:** Mayoritariamente alineado con nivel
**⚠️ Aceptable:** Requiere pequeñas adaptaciones
**❌ Problemático:** No corresponde al nivel o tiene errores

### Criterio 5: Duración apropiada
**✅ Excelente:** 1-3 minutos, óptimo para atención del alumnado
**✅ Bueno:** Duración adecuada
**⚠️ Aceptable:** Un poco largo/corto pero usable
**❌ Problemático:** Demasiado largo (>5 min) o demasiado corto (<30 seg)

---

## 3. MAPAS MENTALES

### Criterio 1: Vocabulario 100% nivel MCER
**✅ Excelente:** Todo el vocabulario es exactamente del nivel objetivo
**✅ Bueno:** 95% del vocabulario del nivel, 1-2 palabras requiere explicación
**⚠️ Aceptable:** 85-90% del nivel, varias palabras requieren explicación
**❌ Problemático:** Menos del 80% del nivel o vocabulario incorrecto

### Criterio 2: Organización lógica
**✅ Excelente:** Categorías semánticas claras, relaciones correctas, estructura intuitiva
**✅ Bueno:** Organización mayoritariamente lógica
**⚠️ Aceptable:** Algunas relaciones cuestionables pero funcional
**❌ Problemático:** Organización confusa o relaciones incorrectas

### Criterio 3: Diseño visual
**✅ Excelente:** Colores diferencian categorías, jerarquías claras, fácil de leer
**✅ Bueno:** Buen diseño visual
**⚠️ Aceptable:** Diseño funcional pero mejorable
**❌ Problemático:** Difícil de leer, confuso, mal organizado

### Criterio 4: Potencial de expansión
**✅ Excelente:** Alumnado puede fácilmente añadir más palabras o ramas
**✅ Bueno:** Expansible con orientación
**⚠️ Aceptable:** Limitadamente expandible
**❌ Problemático:** No diseñado para expansión

### Criterio 5: Utilidad didáctica
**✅ Excelente:** Múltiples usos posibles (estudio, repaso, presentación, evaluación)
**✅ Bueno:** Útil para 2-3 actividades
**⚠️ Aceptable:** Útil para 1 actividad
**❌ Problemático:** Difícilmente usable en clase

## ✅ CHECKLIST DE VERIFICACIÓN

### Antes de usar recursos generados:

**Imágenes:**
- [ ] Revisé culturalmente (no estereotipos)
- [ ] Verifiqué que es apropiada para el nivel
- [ ] La calidad técnica es suficiente
- [ ] Puede usarse en clase con mínimas adaptaciones

**Audios:**
- [ ] Verifiqué que el acento es nativo
- [ ] La velocidad es adecuada al nivel
- [ ] El sonido es limpio
- [ ] El contenido lingüístico es correcto

**Mapas mentales:**
- [ ] Verifiqué todo el vocabulario (100% nivel MCER)
- [ ] La organización es lógica
- [ ] El diseño es claro
- [ ] Es utilizable directamente en clase
