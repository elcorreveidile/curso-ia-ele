## 📊 RÚBRICA (5 criterios × 4 niveles = 20 puntos cada uno)

### Criterio 1: Definición del rol (20 puntos)

| 4 - Excelente (20) | Rol completamente definido con nivel MCER, perfil alumnado, especialización, limitaciones y formato. |
| 3 - Satisfactorio (15-17) | Rol definido con nivel MCER y perfil básico. Falta profundizar en especialización o limitaciones. |
| 2 - Básico (10-13) | Rol definido vagamente sin nivel MCER claro. Poca especificación. |
| 1 - Insuficiente (0-8) | Sin definición de rol o system prompt genérico sin contexto ELE. |

### Criterio 2: Pruebas realizadas (20 puntos)

| 4 - Excelente (20) | 6+ pruebas sistemáticas con documentación de resultados y mejoras. |
| 3 - Satisfactorio (15-17) | 4-5 pruebas básicas con documentación adecuada. |
| 2 - Básico (10-13) | 2-3 pruebas sin documentación clara. |
| 1 - Insuficiente (0-8) | 0-1 pruebas o sin documentación. |

### Criterio 3: Calidad de respuestas (20 puntos)

| 4 - Excelente (20) | Respuestas consistentemente correctas al nivel, con formato apropiado y culturalmente ricas. |
| 3 - Satisfactorio (15-17) | Respuestas correctas 80% del tiempo, nivel mayormente adecuado. |
| 2 - Básico (10-13) | Respuestas adecuadas 50% del tiempo. Nivel inconsistente. |
| 1 - Insuficiente (0-8) | Respuestas pobres o fuera de nivel consistentemente. |

### Criterio 4: Utilidad docente (20 puntos)

| 4 - Excelente (20) | Sistema reutilizable para múltiples unidades, listo para usar con alumnado real. |
| 3 - Satisfactorio (15-17) | Usable con modificaciones menores para diferentes unidades. |
| 2 - Básico (10-13) | Requiere adaptaciones significativas para ser útil. |
| 1 - Insuficiente (0-8) | Difícilmente reutilizable o aplicable. |

### Criterio 5: Documentación (20 puntos)

| 4 - Excelente (20) | Sistema completo + 5+ ejemplos variados + guía de uso + reflexión profunda. |
| 3 - Satisfactorio (15-17) | Sistema + 3-4 ejemplos + documentación básica. |
| 2 - Básico (10-13) | Sistema + 1-2 ejemplos + documentación mínima. |
| 1 - Insuficiente (0-8) | Sin documentación o ejemplos insuficientes. |
