# Apéndice B: Plantillas en blanco
### Marco FRAME y estructuras base para construir tus propios prompts

\newpage

## CÓMO USAR ESTAS PLANTILLAS

Las plantillas de este apéndice son los marcos vacíos del libro: el andamiaje sin el contenido. Están pensadas para que puedas construir tus propios prompts desde cero, adaptados a tu grupo, tus objetivos y tus materiales, sin tener que recordar qué componentes incluir.

Copia la plantilla que necesites, rellena los campos entre corchetes y ajusta las especificaciones a tu situación concreta. Cuanto más detallado sea lo que escribes en cada campo, mejor será el output.

## PLANTILLA 1: FRAME COMPLETO (VERSIÓN GENERAL)

La plantilla base para cualquier prompt de creación de materiales.

```
Actúa como [ROL: tipo de docente / especialista / autor]
para nivel [NIVEL MCER: A1 / A2 / B1 / B2 / C1 / C2].

FORMATO: [Qué debe producir la IA: tipo de texto,
número de ítems, estructura, extensión en palabras]

AUDIENCIA: [Quiénes son tus alumnos: edad, perfil,
contexto, necesidades específicas, L1 si es relevante]

META: [Qué será capaz de hacer el alumno al terminar
la actividad: verbo comunicativo + contenido concreto]

ESPECIFICACIONES:
- [Requisito 1: lo más importante]
- [Requisito 2]
- [Requisito 3]
- [Requisito 4]
- [Requisito 5: si hay más, añadir]

EJEMPLO: [Opcional pero recomendado: muestra un
ítem o fragmento del tipo de output que esperas]
```

## PLANTILLA 2: CREACIÓN DE TEXTO MODELO

Para crear textos de entrada, modelos de escritura o lecturas para el aula.

```
Actúa como autor de textos didácticos para ELE
nivel [NIVEL].

FORMATO: [Tipo de texto] de [N] palabras.

AUDIENCIA: [Descripción del grupo]

META: El texto servirá para [objetivo didáctico:
presentar vocabulario / modelar un género /
activar conocimiento previo / servir de input
para una actividad de comprensión].

ESPECIFICACIONES:
- Tema: [tema concreto]
- Léxico: ajustado al nivel [NIVEL]; los términos
  que superen el nivel aparecen en negrita
- Gramática: [estructuras que aparecen en el texto]
- El texto es [auténtico / literariamente interesante /
  informativo] — no un ejercicio disfrazado de texto
- [Requisito específico del género: narrativo /
  expositivo / argumentativo / dialógico]
- Después del texto: [qué añadir: preguntas de
  comprensión / mapa de estructura / glosario / etc.]
```

## PLANTILLA 3: CREACIÓN DE EJERCICIO GRAMATICAL

Para ejercicios de gramática con enfoque comunicativo.

```
Actúa como profesor de ELE especializado en gramática
comunicativa para nivel [NIVEL].

FORMATO: Ejercicio de [tipo: contraste / transformación /
completar / identificar / producción] con [N] ítems.

AUDIENCIA: [Descripción del grupo]

META: El alumnado será capaz de [usar / distinguir /
comprender] [estructura gramatical] en contexto
de [uso comunicativo: narración / descripción /
argumentación / interacción, etc.].

ESPECIFICACIONES:
- Estructura trabajada: [estructura A] vs. [estructura B]
  (o solo [estructura A])
- Cada ítem incluye un contexto comunicativo breve
  que hace que la respuesta correcta sea evidente
  por el significado, no solo por la forma
- Las situaciones son variadas: [tipos de situación]
- Solucionario con explicación comunicativa
  (no solo la regla gramatical)
- [Requisito adicional si lo hay]
```

## PLANTILLA 4: ACTIVIDAD DE EXPRESIÓN ORAL

Para role-plays, debates, simulaciones y actividades de interacción.

```
Actúa como profesor de ELE especializado en expresión
oral para nivel [NIVEL].

FORMATO: [Tipo de actividad: role-play / debate /
simulación / tarjetas de conversación / información
incompleta] para [N] alumnos / parejas / grupos.

AUDIENCIA: [Descripción del grupo]

META: El alumnado será capaz de [función comunicativa]
usando [estructuras y vocabulario específicos] en
una situación de [contexto].

ESPECIFICACIONES:
- Situación: [descripción de la situación comunicativa]
- Cada tarjeta / posición incluye: [qué información
  tiene cada participante]
- La información asimétrica / la estructura de
  la actividad obliga a [tipo de interacción real]
- Vocabulario útil al pie de cada tarjeta
- Tiempo sugerido: [N] minutos
- [Criterios de éxito si procede]
```

## PLANTILLA 5: ACTIVIDAD DE EXPRESIÓN ESCRITA

Para tareas de escritura con andamiaje y criterios claros.

```
Actúa como profesor de ELE especializado en escritura
para nivel [NIVEL].

FORMATO: Tarea de escritura de [N] palabras con
andamiaje en [N] fases / instrucciones y criterios.

AUDIENCIA: [Descripción del grupo]

META: El alumnado producirá un [tipo de texto:
correo / narración / artículo / descripción, etc.]
sobre [tema] usando [estructuras y vocabulario]
en un registro [formal / informal / semiformal].

ESPECIFICACIONES:
- Tipo de texto: [género textual]
- Destinatario real o simulado: [a quién va dirigido]
- Propósito comunicativo: [para qué sirve el texto]
- Extensión: [N] palabras
- Contenido mínimo: [qué debe incluir]
- Andamiaje: [modelo previo / preguntas guía /
  banco de expresiones / estructura dada]
- Criterios de evaluación: [4-5 criterios]
```

## PLANTILLA 6: RÚBRICA DE EVALUACIÓN

Para crear criterios de evaluación alineados con el objetivo de la tarea.

```
Actúa como especialista en evaluación de ELE para
nivel [NIVEL].

FORMATO: Rúbrica analítica con [N] criterios y
4 niveles de desempeño.

AUDIENCIA: Docente que evalúa la tarea: [descripción
de la tarea que se va a evaluar]

META: La rúbrica evaluará de forma coherente y
transparente [tipo de producción: oral / escrita]
del alumnado de nivel [NIVEL].

ESPECIFICACIONES:
- Criterios: [listar los N criterios relevantes
  para el tipo de tarea]
- 4 niveles: Excelente (4), Competente (3),
  En desarrollo (2), Insuficiente (1)
- Cada celda: descripción concreta del desempeño
  (no solo adjetivos)
- Versión de autoevaluación para el alumno
  (mismos criterios, lenguaje accesible, 1ª persona)
```

## PLANTILLA 7: FEEDBACK INDIVIDUALIZADO

Para generar comentarios de retroalimentación sobre producciones del alumno.

```
Actúa como profesor de ELE especializado en
corrección formativa para nivel [NIVEL].

FORMATO: Comentario de feedback en 3 secciones.

AUDIENCIA: Alumno/a nivel [NIVEL] que ha producido
[tipo de texto / tarea oral].

META: El feedback ayudará al alumno a comprender
qué ha hecho bien, qué puede mejorar y cuál es
su próximo objetivo concreto.

ESPECIFICACIONES:
- Sección 1 (Lo que funciona): 2-3 aspectos
  concretos con cita del texto o descripción
  del comportamiento oral
- Sección 2 (Para mejorar): máximo 3 errores
  o aspectos, priorizados por impacto comunicativo,
  con versión correcta y explicación breve
- Sección 3 (Tu próximo objetivo): 1 solo punto
  de mejora para la próxima producción
- Tono: directo, orientado al futuro, no deprimente

TEXTO / DESCRIPCIÓN DEL ALUMNO:
[PEGAR AQUÍ]
```

## PLANTILLA 8: ADAPTACIÓN DE MATERIAL EXISTENTE

Para transformar materiales que ya tienes para otro nivel, grupo o contexto.

```
Actúa como profesor de ELE especializado en
adaptación de materiales para [nivel / grupo destino].

FORMATO: Versión adaptada del siguiente material
para [descripción del cambio: nivel / grupo /
destreza / contexto cultural].

AUDIENCIA ORIGINAL: [describir el grupo original]
AUDIENCIA DESTINO: [describir el grupo destino]

META: Adaptar el material manteniendo [qué debe
conservarse: objetivo, estructura, nivel, tema]
y cambiando [qué debe cambiar: vocabulario,
ejemplos, formato, extensión, registros].

ESPECIFICACIONES:
- Qué mantener igual: [listado]
- Qué cambiar: [listado con justificación]
- Tabla de cambios al final: qué se modificó y por qué
- La versión adaptada es completa: no solo los cambios

MATERIAL ORIGINAL:
[PEGAR AQUÍ]
```

## PLANTILLA 9: FLUJO ENCADENADO (MULTI-PROMPT)

Para proyectos que requieren varios prompts en secuencia.

```
[PROMPT 1 — Define el marco]
Actúa como diseñador curricular de ELE para nivel [NIVEL].
Genera: [mapa / programa / esquema] de [unidad / curso / secuencia].
Audiencia: [descripción del grupo]
Objetivo: [objetivo comunicativo principal]
Especificaciones: [detalles del marco]

---

[PROMPT 2 — Usa el output del Prompt 1]
Usando el [mapa / programa / esquema] anterior,
genera ahora: [material específico para la fase X].
Mantén coherencia con: [elemento del Prompt 1
que debe conservarse]
Nuevas especificaciones: [lo específico de esta fase]

---

[PROMPT 3 — Siguiente paso]
Tomando como referencia [el texto / la actividad /
el objetivo] generado antes, crea ahora:
[siguiente material en la secuencia]
Especificaciones: [...]

---

[Continuar según sea necesario]
```

## CHECKLIST DE PROMPT ANTES DE EJECUTAR

Antes de enviar cualquier prompt a la IA, comprueba:

- [ ] ¿He especificado el nivel MCER?
- [ ] ¿He descrito al grupo con suficiente detalle?
- [ ] ¿La meta está formulada como lo que el alumno podrá HACER (no como contenido)?
- [ ] ¿El formato es claro? (tipo, extensión, estructura)
- [ ] ¿He incluido al menos 3 especificaciones concretas?
- [ ] ¿He eliminado instrucciones contradictorias?
- [ ] Si el prompt es complejo, ¿he pedido que haga una cosa a la vez?
- [ ] ¿He añadido un ejemplo si el output es poco convencional?

Si alguna respuesta es «no», el prompt puede mejorarse antes de ejecutarlo.
